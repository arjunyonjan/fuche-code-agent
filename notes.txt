FUCHECODE v3.0 — Build Notes

🔥 New Features (v3)

1. Memory
   Save chat history between sessions

2. Model Switcher
   Switch models without editing code

3. Config File
   ~/.fuchecode/config.toml for settings

4. Streaming
   Real-time token-by-token response (true typewriter)

   Code blocks with colors

Memory Implementation — Details:

history.rs — JSONL storage (~/.fuchecode_history.json)

Save — Every Q&A auto-appended with timestamp

Load — Last 10 messages injected into AI context

Commands — /history (show last 10), /clear (wipe all)

Dependencies — Added chrono (timestamps) + dirs (home path)

Files changed: history.rs (new), ollama.rs (history injection), main.rs (commands)

Glitches Fixed (v3.1):

1. popup.items private field — compile error. Added items() / items_len() getters to src/ui/popup.rs
2. Duplicate command sections — 6 near-identical blocks for /clear, /status, /model, /history etc. Consolidated into one popup menu
3. Failed input reader — crossterm event polling crashed with "Failed to initialize input reader". Replaced with simple io::stdin().read_line() loop
4. Unused imports + variable — cleaned up crossterm::event, std::time::Duration, input_buffer
