🚀 Quick Start — ClawRouter
  1. Terminal 1 → npx @blockrun/clawrouter@latest (keep running)
  2. Terminal 2 → cargo run (in fuchecode dir)
  3. In fuchecode → /provider clawrouter
  4. In fuchecode → /model auto
  5. Ask away — tools + chat will work fast

ClawRouter runs on port 8402, free tier. No API key needed.
