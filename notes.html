🚀 RUN IT

Run Locally

Linux / WSL — requires Ollama running with a model loaded.

quick start 
# make executable & run
chmod +x fuchecode.sh
./fuchecode.sh
or directly with cargo 
# clean build & run
cd ~/projects/fuchecode
cargo clean && cargo run

📋 prerequisites

  • Rust toolchain via rustup
  • Ollama serving on localhost:11434
  • A model pulled (e.g. ollama pull qwen2.5:72b)
