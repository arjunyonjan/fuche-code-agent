ClawRouter → FUCHECODE pipeline: Start ClawRouter (port 8402) → point api_url to it → set default_model = blockrun/free → /model toggles cloud ↔ local

🚀 Start ClawRouter — 3 ways port 8402
1 One-shot (recommended)
npx @blockrun/clawrouter@latest
2 Background (survives terminal close)
nohup npx @blockrun/clawrouter@latest > /dev/null 2>&1 &
3 Custom port
npx @blockrun/clawrouter@latest --port 8402
⚙️ FUCHECODE config ~/.fuchecode/config.toml
# ClawRouter (cloud — default)
default_model = "blockrun/free"
api_url = "http://127.0.0.1:8402/v1/chat/completions"
# Ollama (local — toggle with /model)
# default_model = "qwen3-coder:480b-cloud"
# api_url = "http://127.0.0.1:11434/api/chat"
/model blockrun/free ↔ /model qwen3-coder:480b-cloud
