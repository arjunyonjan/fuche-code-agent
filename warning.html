trending

Ollama bound to 127.0.0.1 only

Not accessible from WSL. Ollama needs to listen on 0.0.0.0 so WSL can reach it via the Windows host IP.

🛠️ One-time fix Windows PowerShell as Admin
# Stop Ollama
taskkill /f /im ollama.exe
# Restart with network binding
$env:OLLAMA_HOST="0.0.0.0"
ollama serve
