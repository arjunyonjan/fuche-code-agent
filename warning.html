🐛 Bug Fix Log — Round 1 Jun 12, 2026 · 5 fixes
1.
Empty input loop
Crossterm event::poll(100ms) was spinning + mixing with blocking stdin::read_line(). Rewrote with raw mode, character-by-character buffer, submit on Enter.
2.
Double response output
ollama.rs streamed + printed tokens, then main.rs re-printed whole response char-by-char. Removed duplicate loop, moved color into stream handler.
3.
No streaming feedback
Terminal hung silently while Ollama thought. Added 💭 indicator, overwritten by first token via \r│ .
4.
Dead code warnings
Popup selected field + handle_key unused. Added #[allow(dead_code)].
5.
Write blocked in Plan mode
write_file now checks mode.writes_allowed(). Runs in BUILD, blocked in PLAN.
🔧 Bug Fix Log — Round 2 Jun 12, 2026 · 5 fixes
1.
Prompt misaligned
│ started at column 10 instead of 0. Raw mode \n drops line without carriage return. Added \r after \n in user_prompt_with_mode.
2.
Header wrapping in raw mode
All println! in header lacked \r. Replaced every println! with print!("\r...\n") in ui.rs.
3.
Esc kills terminal
KeyCode::Esc => break closed the app. Removed handler entirely. Only exit or Ctrl+C quits now.
4.
Terminal stuck on crash
Raw mode stayed enabled on panic/Ctrl+C, leaving terminal unusable. Added RawGuard struct with Drop — auto-runs disable_raw_mode() on any exit path.
5.
commands.rs output broken
All println! in command handlers missing \r. Created rl() helper — wraps print!("\r...\n") with flush. Replaced all println! calls.

save() silently drops writes on I/O error. load_last() returns empty vec on any read failure. Corrupted JSONL lines are skipped instead of crashing.

🧒 Like trying on shoes before buying — we checked every button and lever works so you don't find out later it's broken.

search_content("") matches everything. list_dir("") defaults to .. Files with spaces/special chars handled. Invalid paths return "Not found" / "Error: ..." instead of crashing.
